# AI-Learning-Path-Generator

## Overview

The AI Learning Path Generator is a Streamlit-based web application that creates structured weekly learning plans based on user input such as topic, skill level, and duration.

This project was developed using Cursor IDE with AI-assisted coding support.

## Features

- Accepts learning topic from user
- Supports multiple skill levels:
  - Beginner
  - Intermediate
  - Advanced
- Generates weekly structured learning roadmap
- Simple interactive UI using Streamlit
- Demonstrates AI-assisted development workflow

## Technologies Used

- Python
- Streamlit
- Cursor IDE

## How It Works

The user enters:

- Learning topic
- Skill level
- Duration (number of weeks)

The application generates a personalized weekly learning roadmap divided into phases:

- Fundamentals
- Core Skills
- Projects and Mastery

## Example Output

Week 1: Fundamentals  
Week 2: Core Skills  
Week 3: Projects + Mastery
